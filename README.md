# 넌센스 퀴즈 게임 (Nonsense Quiz Game)

## 📋 프로젝트 소개
Socket 통신을 활용한 멀티플레이어 넌센스 퀴즈 게임입니다. 여러 플레이어가 실시간으로 참여하여 재미있는 넌센스 퀴즈를 풀어볼 수 있습니다.

## 🎮 주요 기능

### 1. 메인 화면
- 게임 시작
- 옵션 설정
- 순위표 확인
- 게임 종료

### 2. 옵션 설정
- 폰트 설정
- 폰트 크기 조절
- 윈도우 크기 조절
- 색상 테마 변경
- BGM 설정

### 3. 순위표
- 실시간 플레이어 순위 확인
- 최고 점수 기록
- 승률 통계

### 4. 게임 화면
- 실시간 멀티플레이
- 타이머 기능
- 점수 시스템
- 채팅 기능

### 5. 게임 결과
- 최종 점수 표시
- 승패 결과
- 재시작 옵션
